Acerca del Sistema

Sistema de Gestión de Tesis Digital para la Universidad Nacional José Faustino Sánchez Carrión

Objetivo

Digitalizar y optimizar el proceso de gestión de tesis universitarias, facilitando la colaboración entre estudiantes, docentes y administradores de la UNJFSC.

Visión

Ser la plataforma integral de referencia para la gestión académica de tesis, promoviendo la eficiencia, transparencia y calidad en el proceso educativo superior.

Características Técnicas
Frontend
  • HTML5
  • CSS3 + Bootstrap 5
  • JavaScript
  • Diseño Responsivo
Backend
  • Python 3.8+
  • Flask Framework
  • SQLite Database
  • JWT Authentication
Roles del Sistema
Estudiante
  • Registrar propuesta de tesis
  • Subir documentos
  • Ver estado y comentarios
  • Recibir notificaciones
Docente
  • Revisar tesis asignadas
  • Aprobar/rechazar tesis
  • Agregar comentarios
  • Calificar trabajos
Administrador
  • Gestionar usuarios
  • Asignar asesores/jurados
  • Ver estadísticas
  • Configurar sistema
